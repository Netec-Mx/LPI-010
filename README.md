# Linux Essentials

## Plataforma de laboratorios

Te damos la bienvenida a la **plataforma de laboratorios** del curso **Linux Essentials**. Aquí podrás explorar diferentes tecnologías a través de prácticas guiadas. ¡Desarrolla tus habilidades y lleva tus conocimientos al siguiente nivel!

Curso práctico y accesible que enseña conceptos esenciales de Linux: comandos, sistema de ficheros, permisos, usuarios, procesos y redes básicas, preparándote para tareas iniciales y la certificación Linux Essentials.
